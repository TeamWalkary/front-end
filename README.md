# Walkary Front-End Team

## ⚙ Convention

### commit message

| 헤더     | 내용                                     |
| -------- | ---------------------------------------- |
| feat     | 새로운 기능 추가                         |
| refactor | 코드 리팩토링                            |
| fix      | 버그 수정                                |
| design   | 기능에 지장 가지 않는 스타일 변경        |
| init     | 빌드수행, 패키지 설치, 환경 설정 수정 등 |
| etc      | 주석 추가삭제, reademe 작성              |
